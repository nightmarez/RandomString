# RandomString
Random string builder

```
Generate random alphanumerical strings with special characters:
ONAwjiyQ?D
!##v&U"qZ?
NM59M,v3n)
x?.VUgAr$D
U:;e*tvvFz

Generate random numerical strings:
0067689222
9114135019
8485724729
8039286916
9902973970

Generate unique random strings from limited characters:
112
221
111
122
222
121
212
211

Catch exception when can't more generate unique string:
111
222
122
212
121
221
211
112
Exception: Too many attempts to find unique string.
```
